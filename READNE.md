# HSI-Thesis

An End-to-End Physics-Informed Neural Network Framework for Hyperspectral Image Analysis with Explainable Classification.

## Supervisor
Tanver Ahmed, Assistant Professor, Dept. of CSE, Varendra University

## Features
- Auto band segmentation via correlation analysis
- Metaheuristic band selection (PSO/GWO/MFO) with PINN fitness
- mRMR redundancy reduction
- Traditional ML and CNN classifiers
- Explainable AI (SHAP/LIME)

## Setup
pip install -r requirements.txt

## Datasets
- Indian Pines, Pavia University, Salinas (Earth)
- CRISM (Mars) – auto-download scripts provided
